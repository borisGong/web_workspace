• 使用margin:0 auto实现

    块水平居中
    .child{ width:200px; margin:0 auto; }
    优点：兼容性好 缺点: 需要指定宽度

  • 使用inline-block 和 text-align实现

    块水平居中
    .parent{ text-align: center; } .child{ display: inline-block; }
    优点：兼容性好； 不足：需要同时设置子元素和父元素

  • 使用table实现

    块水平居中
    .child{ display:table; margin:0 auto; }
    优点:只需要对自身进行设置 不足:IE6,7需要调整结构

  • 使用绝对定位实现

    块水平居中
    .parent{ position:relative; } .child{ position:absolute; left:50%; transform:translate(-50%); } /*或者实用margin-left的负值为盒子宽度的一半也可以实现，不过这样就必须知道盒子的宽度，但兼容性好*/
    不足：兼容性差,IE9及以上可用

  • 使用flex实现

    块水平居中
    .parent{ display:flex; justify-content:center; }
    缺点：兼容性差，如果进行大面积的布局可能会影响效率

  • 使用flex实现

    块水平居中
    .parent{ display:flex; } .child{ margin:0 auto; }
    缺点：兼容性差，如果进行大面积的布局可能会影响效率
